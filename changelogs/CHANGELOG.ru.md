# Журнал изменений

## [Unreleased]

## [2.0.3] - 2026-03-19

### Добавлено

- Мягкие напоминания об обновлениях Sparkle: при обнаружении новой версии в фоновом режиме на значке в строке меню появляется красная точка, а рядом с кнопкой шестерёнки и кнопкой «Проверить обновления» в настройках отображается метка «Доступно обновление»
- Если значок в строке меню скрыт, он временно отображается с индикатором при обнаружении обновления и снова скрывается после завершения сеанса

### Изменено

- Окно настроек теперь скрывается, когда Sparkle находит обновление (ранее — только при начале загрузки), и восстанавливается при отмене
- Заголовок окна настроек теперь локализован на всех поддерживаемых языках
- Номер версии перемещён из заголовка настроек в раздел обновлений, рядом с кнопкой «Проверить обновления»

## [2.0.2] - 2026-03-19

### Добавлено

- Кнопка закрытия в строках боковой панели списка окон: при наведении на название окна появляется кнопка ×, позволяющая закрыть окно
- Настройка «Завершать приложение при закрытии последнего окна» (Настройки > Окна): при включении (по умолчанию) закрытие последнего окна приложения завершает его; при отключении закрывается только окно
- Подсказка кнопки закрытия показывает название окна; когда действие завершит приложение, показывается название приложения
- Клавиша «/» для закрытия выбранного окна (или завершения приложения, если это последнее окно и настройка включена)

## [2.0.1] - 2026-03-19

### Изменено

- Панель настроек переработана в стиле Tahoe: секции со стеклянным фоном (Liquid Glass на macOS 26+), компактная панель инструментов с кнопками «Назад» и «Завершить», группированные строки в стиле iOS с встроенными элементами управления

### Исправлено

- Исправлена проблема, при которой размещение окна использовало устаревшую геометрию экрана, когда Dock или строка меню автоматически показывались/скрывались во время отображения оверлея
- Исправлена проблема, при которой позиция или размер окна не применялись, когда целевое приложение асинхронно корректирует свою геометрию после изменения AX; фоновый цикл проверки теперь автоматически повторно применяет отклонившиеся атрибуты

## [2.0.0] - 2026-03-19

### Изменено

- Выпадающее меню выбора целевого окна заменено боковой панелью с Liquid Glass (macOS Tahoe); включает поле поиска с полной поддержкой IME, навигацию клавишами-стрелками и Tab/Shift+Tab, а также Cmd+F для переключения видимости

### Улучшено

- Окна в боковой панели перечислены в Z-порядке (от переднего к заднему), а не сгруппированы по приложениям
- Нестандартные окна (палитры, панели инструментов и т. д.) исключены из списка целевых окон — отображаются только масштабируемые окна документов

## [1.2.7] - 2026-03-18

### Улучшено

- Главное окно автоматически закрывается, когда Sparkle начинает загрузку обновления

### Исправлено

- Устранены видимые швы в предварительном просмотре ограничений изменения размера при одновременном отображении областей переполнения (красные) или недостатка (жёлтые) в обоих направлениях

## [1.2.6] - 2026-03-18

### Исправлено

- При изменении размера фонового окна того же приложения через переключение Tab окно теперь выводится на передний план, если оно было бы скрыто за другими окнами этого приложения

## [1.2.5] - 2026-03-18

### Добавлено

- Обнаружение ограничений изменения размера окон: автоматическое определение возможности изменения размера по осям с помощью быстрой 3-уровневой проверки (неизменяемый размер → кнопка полноэкранного режима → проба 1px как запасной вариант)
- Предварительный просмотр макета теперь показывает красные области, где окно не может быть увеличено, и жёлтые области, где оно не может быть уменьшено, обеспечивая визуальную обратную связь об ограничениях размера перед применением

## [1.2.4] - 2026-03-17

### Улучшено

- Доработан интерфейс редактирования пресетов макетов: кнопка удаления перемещена рядом с кнопкой подтверждения, кнопки редактирования/действий размещены в отдельной колонке, чтобы не перекрывать горячие клавиши
- Выбор области сетки теперь редактируется в режиме редактирования: перетащите по сетке для обновления позиции пресета с предварительным просмотром и подсветкой в реальном времени

## [1.2.3] - 2026-03-17

### Улучшено

- Доработан интерфейс редактирования пресетов макетов: кнопка удаления отображается поверх предварительного просмотра сетки, с диалогом подтверждения, непрозрачным фоном при наведении и единообразным стилем кнопок

## [1.2.2] - 2026-03-17

### Изменено

- Переработано редактирование пресетов макетов для более интуитивного управления настройками

## [1.2.1] - 2026-03-17

## [1.2.0] - 2026-03-17

### Добавлено

- Выпадающее меню целевого окна: нажмите на область информации о цели для выбора окна из всплывающего меню
- Tab и Shift+Tab теперь зарезервированы и не могут быть назначены как горячие клавиши макетов

## [1.1.5] - 2026-03-16

### Добавлено

- Наложение на несколько экранов: окно сетки макета теперь отображается одновременно на всех подключённых экранах
- Размещение между экранами: перетащите сетку или нажмите на пресет на дополнительном экране для размещения целевого окна на этом экране
- Наложение предварительного просмотра отображается на экране, где показано окно пресета

### Исправления

- Исправлен макет «Максимизировать», не заполнявший весь экран при размещении между дисплеями разного размера
- Исправлены локальные сочетания клавиш (клавиши со стрелками, горячие клавиши пресетов), не работавшие после второй активации наложения
- Исправлено закрытие только части окон наложения при нажатии на окно фонового приложения; теперь все окна наложения закрываются одновременно
- Исправлено выделение при наведении/выборе пресета, отображавшееся на всех экранах; теперь отображается только на экране с курсором мыши
