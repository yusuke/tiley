# 변경 이력

## [Unreleased]

## [1.2.2] - 2026-03-17

### 변경

- 레이아웃 프리셋 편집을 보다 직관적으로 재설계

## [1.2.1] - 2026-03-17

### 수정

- 다운로드한 DMG에서 실행 시 "복사" 대신 "이동" 대화상자가 잘못 표시되는 문제 수정 (Gatekeeper App Translocation으로 인해 디스크 이미지 경로가 인식되지 않는 문제)

## [1.2.0] - 2026-03-17

### 추가

- 윈도우 대상 전환: 오버레이가 표시된 상태에서 Tab / Shift+Tab을 눌러 사용 가능한 윈도우를 순환
- 윈도우 대상 드롭다운: 대상 정보 영역을 클릭하여 팝업 메뉴에서 윈도우 선택
- Tab과 Shift+Tab은 예약 키로 지정되어 레이아웃 단축키에 할당 불가

## [1.1.8] - 2026-03-16

### 추가

- DMG에서 복사 후 디스크 이미지 추출 및 DMG 파일 휴지통 이동 제안
- /Applications에서 시작 시 마운트된 Tiley DMG 감지 (Finder에서 수동 복사 후 등) 및 추출과 휴지통 이동 제안

## [1.1.7] - 2026-03-16

### 변경

- 배포 형식을 zip에서 Applications 바로가기 및 사용자 정의 Finder 레이아웃(큰 아이콘, 정사각형 창)이 포함된 DMG로 변경

### 수정

- 다운로드한 zip에서 이동하지 않고 실행 시 읽기 전용 볼륨 오류로 "응용 프로그램으로 이동"이 실패하는 문제 수정 (Gatekeeper App Translocation)
- 디스크 이미지(DMG)에서 실행 시 "이동" 대신 "응용 프로그램으로 복사" 대화상자 표시

## [1.1.6] - 2026-03-16

### 수정

- 다중 화면 설정에서 설정 창을 열려면 두 번 활성화해야 하는 문제 수정 (메뉴 바 아이콘, Cmd+,, Tiley 메뉴 → 설정 모두 영향)

## [1.1.5] - 2026-03-16

### 추가

- 다중 화면 오버레이: 레이아웃 그리드 창이 연결된 모든 화면에 동시에 표시
- 교차 화면 타일링: 보조 화면에서 그리드를 드래그하거나 프리셋을 클릭하여 대상 윈도우를 해당 화면에 타일링
- 프리뷰 오버레이가 호버된 프리셋 윈도우가 표시되는 화면에 표시

### 수정

- 서로 다른 크기의 디스플레이 간 타일링 시 최대화 레이아웃이 전체 화면을 채우지 않는 문제 수정
- 두 번째 오버레이 활성화 후 로컬 키보드 단축키(방향키, 프리셋 핫키)가 작동하지 않는 문제 수정
- 배경 앱 윈도우 클릭 시 일부 오버레이 윈도우만 닫히는 문제 수정; 모든 오버레이 윈도우가 동시에 닫히도록 변경
- 프리셋 호버/선택 하이라이트가 모든 화면에 표시되는 문제 수정; 마우스 커서가 있는 화면에만 표시되도록 변경

## [1.1.4] - 2026-03-15

### 수정

- "Dock 아이콘 표시" 토글이 작동하지 않는 문제 수정: 활성화해도 Dock 아이콘이 나타나지 않고, 비활성화하면 창이 사라짐
- 모든 창을 닫으면 앱이 예기치 않게 종료되는 문제 수정
- 앱을 더블클릭으로 시작 시 윈도우 대상이 Tiley 자체로 기본 설정되는 문제 수정; 이전에 활성화된 앱의 윈도우를 올바르게 대상으로 지정
- 로그인 항목으로 시작 시 메인 창이 표시되는 문제 수정: 시스템 시작 시 자동 실행될 때 더 이상 창이 열리지 않음

## [1.0.0] - 2026-03-14

### 추가

- 다른 위치에서 시작 시 앱을 /Applications로 이동하라는 프롬프트 표시
