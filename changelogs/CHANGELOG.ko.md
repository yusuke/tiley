# 변경 이력

## [Unreleased]

## [3.0.0] - 2026-03-23

### 추가

- 프라이버시를 고려한 사용 통계 수집을 위해 TelemetryDeck SDK 통합 (오버레이 열기, 레이아웃 적용, 프리셋 적용, 설정 변경)
- 사이드바 윈도우를 화면별·앱별로 그룹화. 다중 윈도우 앱은 앱 헤더 아래에 들여쓰기 표시
- 사이드바의 화면 헤더에 케밥 메뉴 추가. "윈도우 모으기" 및 "윈도우를 다른 화면으로 이동" 작업 가능
- 앱 헤더 케밥 메뉴에 "모든 윈도우를 다른 화면으로 이동", "다른 앱 숨기기", "종료" 추가
- 단일 윈도우 앱 케밥 메뉴에 "다른 화면으로 이동", "다른 앱 숨기기", "종료" 추가
- 윈도우가 없는 화면도 사이드바에 화면 헤더 표시

### 변경

- 그리드 배경이 macOS 배경화면 표시 설정(전체 화면 채우기, 화면에 맞게, 늘이기, 가운데, 타일)을 올바르게 반영하도록 개선. 타일 크기 조정, 가운데 모드의 물리적 픽셀 비율, 레터박스 영역의 채우기 색상 지원
- 레이아웃 그리드 미리보기에 메뉴 바, Dock, 노치가 표시되어 실제 화면과 더 유사하게 재현

### 수정

- 리사이즈 전 윈도우 위치가 타겟 위치와 동일할 때 리사이즈 후 윈도우가 의도하지 않은 위치로 이동하는 문제 수정. 사전 넛지를 통한 AX 중복 제거 회피
- 비프라이머리 스크린에서 리사이즈 시 깜빡임 감소. 먼저 현재 위치에서 리사이즈를 시도하고, 완전히 실패한 경우에만 프라이머리 스크린으로 바운스
- 프라이머리 스크린으로 바운스 시 윈도우를 화면 상단(0,0) 대신 화면 최하단(거의 화면 밖)에 배치하여 깜빡임 최소화

## [2.2.0] - 2026-03-21

### 변경

- 선택되지 않은 그리드 타일을 투명하게 변경
- 그리드 가로세로 비율을 메뉴 바 및 Dock을 제외한 화면 크기에 맞게 조정. 그리드가 너무 높아질 경우 비율을 유지하면서 너비를 축소하여 최소 4개의 프리셋이 항상 표시되도록 개선
- 레이아웃 그리드 배경에 데스크탑 사진 표시 (반투명, 모서리 둥글게)
- 드래그 선택 중인 셀을 반투명으로 변경하여 배경 데스크탑 사진이 비쳐 보이도록 개선
- 그리드 내 프리셋 호버 하이라이트를 드래그 선택 시와 동일한 스타일로 통일

### 추가

- 멀티 모니터 환경에서 윈도우 목록 사이드바를 대상 화면뿐만 아니라 모든 화면에 표시
- 사이드바 상태(표시/숨김, 선택된 항목, 검색 텍스트)를 모든 화면의 윈도우 간에 동기화
- 리사이즈 디버그 로그 출력 옵션 (`~/tiley.log`) 추가 (설정 > 디버그)

### 수정

- 오버레이 표시 중 Dock 또는 메뉴 막대가 자동으로 표시/숨김되면 이전 화면 정보를 사용하여 윈도우가 잘못된 위치에 배치되는 문제를 수정
- DPI가 다른 디스플레이 구성에서 비프라이머리 스크린의 윈도우가 리사이즈되지 않는 문제를 수정. 리사이즈 시 윈도우를 일시적으로 프라이머리 디스플레이로 이동한 후 목표 위치에 배치
- 리사이즈 시 일부 앱이 윈도우 위치를 원래대로 되돌려 지정한 위치로 이동되지 않는 문제를 수정 (AX 중복 제거 우회)
- 앱의 최소 윈도우 크기로 인해 지정 크기까지 축소할 수 없을 경우, 화면 밖으로 나가지 않도록 윈도우 위치를 자동으로 재계산
- 대상 윈도우를 다른 화면의 윈도우로 전환할 때 윈도우가 순간적으로 깜빡이는 문제를 수정 (화면 변경 시 윈도우 재생성 폐지)

## [2.1.0] - 2026-03-20

### 추가

- 사이드바 창 목록에서 더블 클릭하면 해당 창을 최전면으로 가져오고 레이아웃 그리드를 닫음
- 사이드바 창 행의 컨텍스트 메뉴(말줄임표 버튼)에 세 가지 동작 추가:
  - "[앱]의 다른 창 닫기" — 같은 앱의 다른 창을 닫음 (앱에 여러 창이 있을 때만 표시)
  - "[앱] 종료" — 애플리케이션을 종료
  - "[앱] 외 창 숨기기" — 다른 모든 애플리케이션을 숨김 (Cmd-H 동작), 선택한 앱이 숨겨져 있으면 숨김 해제
- 숨겨진 (Cmd-H) 애플리케이션이 사이드바에 플레이스홀더 항목으로 표시되며 (앱 이름만) 50% 불투명도로 표시
- 숨겨진 앱을 선택하면 (Enter, 더블 클릭, 그리드/레이아웃 리사이즈) 자동으로 숨김 해제되고 해당 앱의 최전면 창에 대해 작동

## [1.0.0] - 2026-03-14

### 추가

- 다른 위치에서 시작 시 앱을 /Applications로 이동하라는 프롬프트 표시
- 단축키별 글로벌 플래그: 레이아웃 프리셋 내 각 단축키를 개별적으로 글로벌 또는 로컬로 설정 가능
- 일반 단축키와 글로벌 단축키용 별도 추가 버튼(인스턴트 팝오버 도구 설명 포함)

### 변경

- 글로벌 단축키 설정을 프리셋 단위에서 단축키 단위로 변경
- 레거시 프리셋 단위 글로벌 플래그가 있는 기존 프리셋은 자동으로 마이그레이션

## [0.9.0] - 2026-03-14

- 최초 릴리스

### 추가

- 사용자 정의 가능한 그리드 크기로 윈도우 타일링을 위한 그리드 오버레이
- 글로벌 키보드 단축키(Shift + Command + Space)로 오버레이 활성화
- 그리드 셀을 드래그하여 대상 윈도우 영역 정의
- 윈도우 배치 저장 및 복원을 위한 레이아웃 프리셋
- 다중 디스플레이 지원
- 로그인 시 시작 옵션
- 현지화: 영어, 일본어, 한국어, 중국어 간체, 중국어 번체
