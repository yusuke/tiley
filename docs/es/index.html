Una utilidad macOS gratuita y sencilla para organizar tus ventanas.
Arrastra una cuadrícula, coloca una ventana.

Funciones

▦

Cuadrícula superpuesta

Una cuadrícula personalizable aparece sobre tu pantalla. Arrastra para seleccionar la región donde quieres colocar una ventana.

⌘

Atajos globales

Activa la cuadrícula o aplica diseños predefinidos al instante con atajos de teclado personalizables.

◫

Diseños predefinidos

Guarda diseños frecuentes — mitad izquierda, mitad derecha, maximizar y más — y aplícalos en un solo paso.



Nativo en Apple Silicon

Compilado de forma nativa para Apple Silicon con Swift y AppKit. Rápido, eficiente y hecho para tu Mac.

⚙

Cuadrícula personalizable

Ajusta filas, columnas y el espacio entre celdas para adaptarlo a tu flujo de trabajo y configuración de pantalla.

↻

Actualizaciones automáticas

La comprobación de actualizaciones integrada mediante Sparkle te mantiene en la última versión automáticamente.

Cómo funciona

Activar

Pulsa el atajo global o haz clic en el icono de la barra de menús para mostrar la cuadrícula de diseño.

Seleccionar

Arrastra sobre las celdas de la cuadrícula para definir la región donde quieres colocar tu ventana.

Listo

La ventana en primer plano se mueve y redimensiona para ajustarse a tu selección al instante.

Atajo predeterminado

Abre la cuadrícula con una sola pulsación de teclas.

⇧ Shift + ⌘ Command + Space

FAQ

¿Qué versiones de macOS son compatibles?

Tiley requiere macOS 14 (Sonoma) o posterior.

¿Por qué Tiley necesita permisos de accesibilidad?

macOS requiere acceso de accesibilidad para mover y redimensionar ventanas de otras aplicaciones. Tiley usa la API de accesibilidad únicamente para la gestión de ventanas — no se recopila ningún dato.

¿Puedo cambiar el tamaño de la cuadrícula?

Sí. Abre los ajustes desde la barra de menús para modificar el número de filas, columnas y el espacio entre celdas.

¿Puedo asignar atajos a diseños específicos?

Sí. Cada diseño predefinido (Maximizar, Mitad izquierda, etc.) puede tener su propio atajo global asignado en los ajustes.

¿Tiley es gratuito?

Sí, Tiley es gratuito y de código abierto. El código fuente está disponible en GitHub.
