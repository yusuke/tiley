Tiley

Лёгкая утилита для компоновки окон в macOS. Перетащите сетку, разместите окно.

Скачать GitHub

Выберите область на сетке, чтобы расположить окно.

Возможности

▦

Наложение сетки

Настраиваемая сетка появляется поверх экрана. Перетащите, чтобы выбрать область для размещения окна.

⌘

Глобальные сочетания клавиш

Вызывайте сетку или применяйте предустановки компоновки мгновенно с помощью настраиваемых сочетаний клавиш.

◫

Предустановки компоновки

Сохраняйте часто используемые компоновки — левая половина, правая половина, развернуть и другие — и применяйте их в одно действие.



Нативно на Apple Silicon

Создано нативно для Apple Silicon на Swift и AppKit. Быстро, эффективно и идеально для вашего Mac.

⚙

Настраиваемая сетка

Настройте количество строк, столбцов и отступ между ячейками под ваш рабочий процесс и конфигурацию дисплея.

↻

Автоматические обновления

Встроенная проверка обновлений через Sparkle автоматически поддерживает актуальную версию.

Как это работает

Активировать

Нажмите глобальное сочетание клавиш или кликните на значок в строке меню, чтобы показать сетку компоновки.

Выбрать

Перетащите по ячейкам сетки, чтобы определить область для размещения окна.

Готово

Окно на переднем плане мгновенно перемещается и изменяет размер в соответствии с вашим выбором.

Сочетание клавиш по умолчанию

Откройте наложение сетки одним нажатием.

⇧ Shift + ⌘ Command + Space

FAQ

Какие версии macOS поддерживаются?

Tiley требует macOS 14 (Sonoma) или новее.

Почему Tiley нужны разрешения универсального доступа?

macOS требует доступ к универсальному доступу для перемещения и изменения размера окон других приложений. Tiley использует API универсального доступа исключительно для управления окнами — никакие данные не собираются.

Можно ли изменить размер сетки?

Да. Откройте настройки из строки меню, чтобы изменить количество строк, столбцов и отступ между ячейками.

Можно ли назначить сочетания клавиш для определённых компоновок?

Да. Каждой предустановке компоновки (Развернуть, Левая половина и т. д.) можно назначить собственное глобальное сочетание клавиш в настройках.

Tiley бесплатный?

Да, Tiley бесплатный и с открытым исходным кодом. Исходный код доступен на GitHub.
