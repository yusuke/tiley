Tiley

Ein einfaches macOS-Dienstprogramm zum Anordnen deiner Fenster. Raster ziehen, Fenster platzieren.

Herunterladen GitHub Sponsern

Wähle einen Bereich im Raster, um dein Fenster anzuordnen.

Funktionen

▦

Raster-Overlay

Ein anpassbares Raster erscheint über deinem Bildschirm. Ziehe, um den Bereich auszuwählen, in dem du ein Fenster platzieren möchtest.

⌘

Globale Tastenkürzel

Zeige das Raster an oder wende Layout-Vorlagen sofort mit anpassbaren Tastenkürzeln an.

◫

Layout-Vorlagen

Speichere häufig verwendete Layouts — linke Hälfte, rechte Hälfte, maximieren und mehr — und wende sie in einem Schritt an.



Nativ auf Apple Silicon

Nativ für Apple Silicon mit Swift und AppKit entwickelt. Schnell, effizient und perfekt auf deinem Mac.

⚙

Anpassbares Raster

Passe Zeilen, Spalten und den Abstand zwischen Zellen an deinen Arbeitsablauf und deine Bildschirmeinrichtung an.

↻

Automatische Updates

Die integrierte Update-Prüfung über Sparkle hält dich automatisch auf dem neuesten Stand.

So funktioniert es

Aktivieren

Drücke das globale Tastenkürzel oder klicke auf das Menüleistensymbol, um das Layoutraster anzuzeigen.

Auswählen

Ziehe über die Rasterzellen, um den Bereich zu definieren, in dem dein Fenster platziert werden soll.

Fertig

Das vorderste Fenster wird sofort verschoben und in der Größe angepasst.

Standard-Tastenkürzel

Öffne das Raster-Overlay mit einem Tastendruck.

⇧ Shift + ⌘ Command + Space

FAQ

Welche macOS-Versionen werden unterstützt?

Tiley erfordert macOS 14 (Sonoma) oder neuer.

Warum benötigt Tiley Bedienungshilfen-Berechtigungen?

macOS erfordert Zugriff auf Bedienungshilfen, um Fenster anderer Anwendungen zu verschieben und in der Größe zu ändern. Tiley verwendet die Bedienungshilfen-API ausschließlich zur Fensterverwaltung — es werden keine Daten erfasst.

Kann ich die Rastergröße ändern?

Ja. Öffne die Einstellungen über die Menüleiste, um die Anzahl der Zeilen, Spalten und den Abstand zwischen den Zellen anzupassen.

Kann ich bestimmten Layouts Tastenkürzel zuweisen?

Ja. Jede Layout-Vorlage (Maximieren, Linke Hälfte usw.) kann in den Einstellungen ein eigenes globales Tastenkürzel erhalten.

Ist Tiley kostenlos?

Ja, Tiley ist kostenlos und Open Source. Der Quellcode ist auf GitHub verfügbar.
