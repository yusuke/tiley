FAQ

支持哪些 macOS 版本？

Tiley 需要 macOS 14（Sonoma）或更高版本。

为什么需要辅助功能权限？

macOS 要求辅助功能权限才能移动和调整其他应用的窗口大小。Tiley 仅将辅助功能 API 用于窗口管理，不会收集任何数据。

可以更改网格大小吗？

可以。从菜单栏打开设置，调整行数、列数和单元格间距。

可以为特定布局分配快捷键吗？

可以。每个布局预设（最大化、左半屏等）都可以在设置中分配独立的全局快捷键。

Tiley 是免费的吗？

是的，Tiley 是免费开源软件。源代码可在 GitHub 上获取。

Tiley 会收集个人数据吗？

不会。Tiley 使用 TelemetryDeck 收集匿名使用统计数据，例如应用启动频率和使用了哪些功能。TelemetryDeck 不收集个人数据、IP 地址或设备标识符。所有数据均为汇总数据，无法追溯到个人用户。详情请参阅 TelemetryDeck 隐私政策。

Tiley 的灵感来自哪里？

Tiley 基于网格的窗口布局方式受到了我们多年来一直喜爱使用的窗口管理工具 Divvy 的启发。当我们需要一个原生支持 Apple Silicon 和现代 macOS 的版本时，决定以免费开源应用的形式从头重新构建这一体验。
