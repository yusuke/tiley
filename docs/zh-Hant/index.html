使用方法

啟用

按下全域快捷鍵或點擊選單列圖示，顯示版面網格。

選擇

在網格上拖曳，定義要放置視窗的區域。

完成

最前景的視窗將立即移動並調整大小以適應您的選擇。

預設快捷鍵

一鍵開啟網格覆蓋層。

⇧ Shift + ⌘ Command + Space

FAQ

支援哪些 macOS 版本？

Tiley 需要 macOS 14（Sonoma）或更高版本。

為什麼需要輔助使用權限？

macOS 要求輔助使用權限才能移動和調整其他應用程式的視窗大小。Tiley 僅將輔助使用 API 用於視窗管理，不會收集任何資料。

可以更改網格大小嗎？

可以。從選單列開啟設定，調整列數、欄數和儲存格間距。

可以為特定版面指定快捷鍵嗎？

可以。每個版面預設（最大化、左半邊等）都可以在設定中指定獨立的全域快捷鍵。

Tiley 是免費的嗎？

是的，Tiley 是免費開源軟體。原始碼可在 GitHub 上取得。
