Como funciona

Ativar

Pressione o atalho global ou clique no ícone da barra de menus para exibir a grade de layout.

Selecionar

Arraste sobre as células da grade para definir a região onde deseja posicionar sua janela.

Pronto

A janela em primeiro plano é movida e redimensionada para se ajustar à sua seleção instantaneamente.

Atalho padrão

Abra a grade com uma única combinação de teclas.

⇧ Shift + ⌘ Command + Space

FAQ

Quais versões do macOS são compatíveis?

O Tiley requer macOS 14 (Sonoma) ou posterior.

Por que o Tiley precisa de permissões de acessibilidade?

O macOS exige acesso de acessibilidade para mover e redimensionar janelas de outros aplicativos. O Tiley usa a API de acessibilidade apenas para gerenciamento de janelas — nenhum dado é coletado.

Posso alterar o tamanho da grade?

Sim. Abra os ajustes pela barra de menus para modificar o número de linhas, colunas e o espaçamento entre as células.

Posso atribuir atalhos a layouts específicos?

Sim. Cada layout predefinido (Maximizar, Metade esquerda, etc.) pode ter seu próprio atalho global atribuído nos ajustes.

O Tiley é gratuito?

Sim, o Tiley é gratuito e de código aberto. O código-fonte está disponível no GitHub.
