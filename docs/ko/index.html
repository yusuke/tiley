기능

▦

그리드 오버레이

사용자 지정 가능한 그리드가 화면에 표시됩니다. 드래그하여 창을 배치할 영역을 선택하세요.

⌘

전역 단축키

사용자 지정 가능한 키보드 단축키로 그리드를 표시하거나 레이아웃 프리셋을 즉시 적용할 수 있습니다.

◫

레이아웃 프리셋

왼쪽 절반, 오른쪽 절반, 최대화 등 자주 사용하는 레이아웃을 저장하고 한 번에 적용하세요.



Apple Silicon 네이티브

Swift와 AppKit으로 Apple Silicon에 맞게 네이티브 빌드. 빠르고 효율적이며 Mac에 최적화되어 있습니다.

⚙

사용자 지정 그리드

행, 열 수와 셀 간격을 조정하여 작업 흐름과 디스플레이에 맞출 수 있습니다.

↻

자동 업데이트

Sparkle를 통한 내장 업데이트 확인 기능으로 항상 최신 버전을 자동으로 유지합니다.

사용 방법

실행

전역 단축키를 누르거나 메뉴 막대 아이콘을 클릭하여 레이아웃 그리드를 표시합니다.

선택

그리드 셀을 드래그하여 창을 배치할 영역을 정의합니다.

완료

최전면 창이 선택한 영역에 맞게 즉시 이동 및 크기 조절됩니다.

기본 단축키

키 한 번으로 그리드 오버레이를 엽니다.

⇧ Shift + ⌘ Command + Space

FAQ

어떤 macOS 버전을 지원하나요?

Tiley는 macOS 14(Sonoma) 이상이 필요합니다.

왜 손쉬운 사용 권한이 필요한가요?

macOS에서는 다른 앱의 창을 이동하고 크기를 변경하려면 손쉬운 사용 권한이 필요합니다. Tiley는 창 관리만을 위해 손쉬운 사용 API를 사용하며, 데이터를 수집하지 않습니다.

그리드 크기를 변경할 수 있나요?

네. 메뉴 막대에서 설정을 열어 행, 열 수와 셀 간격을 조정할 수 있습니다.

특정 레이아웃에 단축키를 지정할 수 있나요?

네. 각 레이아웃 프리셋(최대화, 왼쪽 절반 등)에 설정에서 개별 전역 단축키를 지정할 수 있습니다.

Tiley는 무료인가요?

네, Tiley는 무료 오픈 소스입니다. 소스 코드는 GitHub에서 확인할 수 있습니다.
